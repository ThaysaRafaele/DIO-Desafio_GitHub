O Git Bash é um terminal extendido para otimizar o uso do Git

git status				-> Exibe quais alterações estão despreparadas e quais arquivos não estão sendo monitorados pelo Git.
git add . 				-> adiciona os itens na pasta
git commit -m "texto do commit" 	-> (comita com uma mensagem de anotação)
git push origin main			-> envia todos os commits locais para a nuvem (servidor)
